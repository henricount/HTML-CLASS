LESSON 2

Text Formatting & Links

Make your text stand out and connect your pages together

Text Formatting Tags

Just like formatting text in Microsoft Word, HTML allows you to make text bold, italic, and more. These tags help emphasize important information in your articles.

Bold and Strong

There are two ways to make text bold:

Tag Code Example When to Use
<strong> <strong>Important!</strong> For important text (search engines pay attention to this)
<b> <b>Bold text</b> For bold styling without extra importance
For Journalists: Use <strong> for key facts, names, or quotes you want to emphasize. Use <b> just for visual styling.

Italic and Emphasis

Similarly, there are two ways to make text italic:

Tag Code Example When to Use
<em> <em>emphasized text</em> For emphasis (like vocal stress when reading aloud)
<i> <i>italic text</i> For technical terms, foreign words, book titles

Other Useful Formatting Tags

Tag Purpose Example
<u> Underline text <u>underlined</u>
<mark> Highlight text (like a yellow marker) <mark>highlighted</mark>
<small> Smaller text (for fine print) <small>small text</small>
<del> Deleted text (strikethrough) <del>removed</del>
<ins> Inserted text (underlined) <ins>added</ins>

Lists: Organizing Information

Lists are essential for journalism - perfect for bullet points, numbered steps, or quick facts.

Unordered Lists (Bullet Points)

Use <ul> (unordered list) for items where order doesn't matter:

Code
<h3>Key Points</h3>
<ul>
    <li>First point</li>
    <li>Second point</li>
    <li>Third point</li>
</ul>
Output

Key Points

  • First point
  • Second point
  • Third point

Ordered Lists (Numbered Lists)

Use <ol> (ordered list) when sequence matters:

Code
<h3>Steps to Follow</h3>
<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>
Output

Steps to Follow

  1. First step
  2. Second step
  3. Third step
Remember: <li> stands for "list item" and must always be inside either <ul> or <ol> tags.

Nested Lists

You can put lists inside lists for sub-points:

Nested List Example
<h3>Story Elements</h3>
<ul>
    <li>Introduction
        <ul>
            <li>Lead paragraph</li>
            <li>Hook</li>
        </ul>
    </li>
    <li>Body
        <ul>
            <li>Facts</li>
            <li>Quotes</li>
        </ul>
    </li>
    <li>Conclusion</li>
</ul>

Creating Links

Links are what make the web "web-like" - they connect pages together. The <a> tag (anchor) creates hyperlinks.

Basic Link Structure

Link Syntax
<a href="destination.html">Click here</a>

Breaking it down:

  • <a> - the anchor tag
  • href - "hypertext reference" - where the link goes
  • "destination.html" - the file or URL to link to
  • Click here - the clickable text (what users see)
  • </a> - closing tag

Types of Links

Link Type Example Use Case
External Link
(to another website)		 <a href="https://www.bbc.com">BBC News</a> Linking to sources, references
Internal Link
(to another page on your site)		 <a href="about.html">About Us</a> Navigation between your pages
Email Link <a href="mailto:tips@news.com">Email Us</a> Contact information
Same Page Link
(anchor within page)		 <a href="#section2">Jump to Section 2</a> Table of contents, back to top

Opening Links in New Tabs

Use the target="_blank" attribute to open links in a new tab:

New Tab Link
<a href="https://www.nytimes.com" target="_blank">New York Times</a>
Best Practice: Use target="_blank" for external links (keeps users on your site). Let internal links open in the same tab.

Link Titles (Tooltips)

Add a title attribute to show a tooltip when users hover:

Link with Tooltip
<a href="article.html" title="Read the full story">Continue reading</a>

Line Breaks and Special Characters

Line Breaks

Sometimes you need a line break without starting a new paragraph. Use <br>:

Line Break Example
<p>
John Smith<br>
Senior Reporter<br>
Campus News Daily
</p>
Note: <br> is a self-closing tag - it doesn't need a closing </br>. You can also write it as <br />.

Horizontal Lines

Create a horizontal line to separate content with <hr>:

Horizontal Line
<hr>

Special Characters

Some characters have special meaning in HTML, so we use "character entities" to display them:

Character HTML Code Description
< &lt; Less than
> &gt; Greater than
& &amp; Ampersand
" &quot; Quotation mark
© &copy; Copyright symbol
  &nbsp; Non-breaking space

Complete Example: News Article with Links

CREATE A MULTI-PAGE NEWS SITE

Let's create two linked pages - a homepage and an article page.

1Create index.html (Homepage)

<!DOCTYPE html>
<html>
<head>
    <title>Campus News Today</title>
</head>
<body>
    <h1>Campus News Today</h1>
    <hr>

    <h2>Top Stories</h2>

    <h3><a href="library-article.html">New Library Opens
    Next Month</a></h3>
    <p>The long-awaited student library will finally open its
    doors in October. <a href="library-article.html">Read
    more</a></p>

    <h3>Other Headlines</h3>
    <ul>
        <li><strong>Sports:</strong> Basketball team
        wins championship</li>
        <li><strong>Campus Life:</strong> New food
        trucks arrive</li>
        <li><strong>Academics:</strong> Professor
        wins national award</li>
    </ul>

    <hr>
    <p><small>&copy; 2025 Campus News Today</small></p>
</body>
</html>

2Create library-article.html (Article Page)

<!DOCTYPE html>
<html>
<head>
    <title>New Library Opens - Campus News Today</title>
</head>
<body>
    <p><a href="index.html">&lt; Back to Homepage</a></p>

    <h1>New Library Opens Next Month</h1>
    <p><em>By Sarah Johnson | September 15, 2025</em></p>
    <hr>

    <p><strong>Students will soon have access to a
    state-of-the-art library</strong> featuring study spaces,
    computer labs, and over 50,000 books.</p>

    <h2>What to Expect</h2>
    <p>The new facility includes:</p>
    <ul>
        <li>24/7 study areas</li>
        <li>Silent reading rooms</li>
        <li>Group collaboration spaces</li>
        <li>Coffee shop on first floor</li>
    </ul>

    <h2>Opening Schedule</h2>
    <ol>
        <li><strong>October 1:</strong> Soft opening
        for faculty</li>
        <li><strong>October 7:</strong> Grand
        opening for students</li>
        <li><strong>October 14:</strong> Community
        open house</li>
    </ol>

    <p>For more information, visit the
    <a href="https://www.example.com" target="_blank">
    university website</a>.</p>

    <hr>
    <p><a href="index.html">&lt; Back to Homepage</a></p>
</body>
</html>

3Save Both Files

  • Save them in the same folder
  • Open index.html in your browser
  • Click the links to navigate between pages!
Congratulations! You just created a multi-page website with navigation!

Key Takeaways

  • Use <strong> for important text, <em> for emphasis
  • <ul> creates bullet lists, <ol> creates numbered lists
  • Lists need <li> (list item) tags inside them
  • Links use <a href="destination">text</a>
  • Use target="_blank" to open links in new tabs
  • <br> creates line breaks, <hr> creates horizontal lines
  • Special characters use codes like &copy; for ©
