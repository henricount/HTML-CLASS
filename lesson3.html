Images & Multimedia

Add visual elements to make your stories more engaging

Adding Images to Your Webpage

Images are crucial for online journalism. They grab attention, illustrate stories, and break up text. In HTML, we use the <img> tag.

Basic Image Syntax

Image Tag
<img src="photo.jpg" alt="Description of image">

Important parts:

  • <img> - the image tag (self-closing, no </img> needed)
  • src - "source" - the path to your image file
  • alt - "alternative text" - describes the image (essential!)
Why ALT text matters:
  • Screen readers use it for visually impaired users (accessibility)
  • Shows if the image fails to load
  • Helps search engines understand your content
  • Legal requirement in many countries

Image Attributes

Attribute Purpose Example
src Image file location (required) src="photo.jpg"
alt Description of image (required) alt="Students in library"
width Width in pixels width="600"
height Height in pixels height="400"
title Tooltip on hover title="Photo by John Doe"
Tip: Set only width OR height, not both. The browser will automatically scale the other dimension to keep the image proportions correct.

Where to Put Your Images

You have several options for where your image files can be located:

1. Same Folder (Recommended for Beginners)

Image in Same Folder
<img src="photo.jpg" alt="Campus building">

Your HTML file and image are in the same folder.

2. Subfolder (Better Organization)

Image in Subfolder
<img src="images/photo.jpg" alt="Campus building">

Create an "images" folder and put your photos there.

3. Online Image (External URL)

External Image
<img src="https://www.example.com/photo.jpg" alt="Campus building">
Copyright Warning: Only use images you have permission to use! For journalism:
  • Your own photos
  • Photos with Creative Commons licenses
  • Stock photo sites (free or paid)
  • Wire service photos (if your organization subscribes)
  • Photos from sources who grant permission

Free Image Resources for Students

  • Unsplash - Free high-quality photos
  • Pexels - Free stock photos and videos
  • Pixabay - Free images and videos
  • Wikimedia Commons - Free media files

Image Captions and Credits

In journalism, you should always credit photos and add captions. HTML5 has special tags for this:

Figure with Caption
<figure>
    <img src="protest.jpg" alt="Students marching with signs"
         width="600">
    <figcaption>Students march across campus demanding climate
    action. Photo by Sarah Johnson.</figcaption>
</figure>

Breaking it down:

  • <figure> - groups an image with its caption
  • <img> - the actual image
  • <figcaption> - the caption and credit
Best Practice: Always include:
  • What's happening in the image
  • When it was taken (if relevant)
  • Photo credit (who took it)

Common Image Formats

Format Best For Pros Cons
JPG/JPEG Photos, news images Small file size, widely supported Lossy compression
PNG Graphics, logos, screenshots Supports transparency, lossless Larger file size
GIF Simple animations Small, supports animation Limited colors (256)
WebP Modern web images Great compression, quality Older browsers may not support
For Journalism: Use JPG for photographs and PNG for graphics like infographics or charts.

Embedding Videos

Videos are powerful storytelling tools. Most journalists embed videos from YouTube, Vimeo, or other platforms.

Embedding YouTube Videos

1Find the video on YouTube

2Click "Share" below the video

3Click "Embed"

4Copy the code

YouTube gives you code like this:

YouTube Embed Code
<iframe width="560" height="315"
src="https://www.youtube.com/embed/VIDEO_ID"
title="YouTube video player"
frameborder="0"
allow="accelerometer; autoplay; clipboard-write;
encrypted-media; gyroscope; picture-in-picture"
allowfullscreen></iframe>

Just paste this code directly into your HTML where you want the video!

HTML5 Video (Your Own Video Files)

If you have your own video file, use the <video> tag:

HTML5 Video
<video width="640" height="360" controls>
    <source src="interview.mp4" type="video/mp4">
    <source src="interview.webm" type="video/webm">
    Your browser doesn't support video playback.
</video>

Attributes explained:

  • controls - shows play/pause buttons
  • autoplay - starts playing automatically (use sparingly!)
  • loop - repeats the video
  • muted - starts with sound off
  • poster="image.jpg" - thumbnail before playing
Important: Video files are large! For journalism websites, it's usually better to upload videos to YouTube or Vimeo and embed them, rather than hosting them yourself.

Embedding Audio

Perfect for podcasts, interviews, or radio-style journalism.

HTML5 Audio
<audio controls>
    <source src="interview.mp3" type="audio/mpeg">
    <source src="interview.ogg" type="audio/ogg">
    Your browser doesn't support audio playback.
</audio>

Adding Audio with Description

Audio with Context
<h3>Listen to the Interview</h3>
<p>Reporter Jane Smith speaks with the mayor about
the new budget proposal.</p>
<audio controls>
    <source src="mayor-interview.mp3" type="audio/mpeg">
    Your browser doesn't support audio playback.
</audio>
<p><small>Interview conducted May 15, 2025 | Duration:
8:32</small></p>

Complete Example: Multimedia Article

CREATE A MULTIMEDIA STORY

1Create Your Project Folder

  • Make a new folder called "multimedia-story"
  • Inside it, create a folder called "images"

2Download a Free Image

  • Go to Unsplash.com
  • Download any image you like
  • Save it in your "images" folder
  • Rename it to something simple like "photo1.jpg"

3Create multimedia-article.html

<!DOCTYPE html>
<html>
<head>
    <title>Climate Action Rally Draws Thousands</title>
</head>
<body>
    <h1>Climate Action Rally Draws Thousands to Capitol</h1>
    <p><em>By Alex Rivera | October 15, 2025</em></p>
    <hr>

    <p><strong>Over 5,000 people gathered today</strong>
    to demand action on climate change in what organizers call
    the largest environmental protest in state history.</p>

    <figure>
        <img src="images/photo1.jpg"
             alt="Large crowd of protesters with signs"
             width="700">
        <figcaption>Protesters gather outside the state
        capitol building. Photo by Alex Rivera.</figcaption>
    </figure>

    <h2>What Protesters Want</h2>
    <p>Demonstrators are calling for:</p>
    <ul>
        <li>100% renewable energy by 2035</li>
        <li>Ban on new fossil fuel projects</li>
        <li>Investment in green jobs</li>
        <li>Climate education in schools</li>
    </ul>

    <h2>Student Voices</h2>
    <p>"Our future is at stake," said Maria Chen, 19, a
    university student. "We can't wait any longer for action."</p>

    <h3>Watch: Student Leader Speaks</h3>
    <iframe width="560" height="315"
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
    title="Student speech at climate rally"
    frameborder="0" allowfullscreen></iframe>
    <p><small>Note: Replace with your own video</small></p>

    <h2>What's Next</h2>
    <p>Organizers plan to meet with lawmakers next week to
    present their demands. A follow-up rally is scheduled
    for next month.</p>

    <hr>
    <p><small>&copy; 2025 Campus News Network</small></p>
</body>
</html>

4Test Your Page

  • Save the HTML file in your "multimedia-story" folder
  • Open it in your browser
  • Your image should appear!
Excellent! You've created a multimedia news article with images and embedded video!

Best Practices for Journalism

Image Best Practices

  • Always write descriptive ALT text - "Student protesters" not "image1"
  • Credit every photo - photographer name and date
  • Optimize image size - large files slow down pages
  • Use relevant images - directly related to your story
  • Caption everything - explain what readers are seeing

Video Best Practices

  • Keep it short - 1-3 minutes ideal for web
  • Add context - headline and description before the video
  • Include transcripts - for accessibility and SEO
  • Test on mobile - many readers watch on phones

Copyright Checklist

Before using any image or video, ask:
  1. Did I create this content?
  2. Do I have written permission to use it?
  3. Is it licensed for reuse (Creative Commons)?
  4. Is it in the public domain?

If you answered "no" to all four, don't use it!

Troubleshooting Images

Problem Likely Cause Solution
Image doesn't show Wrong file path Check spelling and folder location
Broken image icon File doesn't exist Verify file is in correct folder
Image too big/small No size specified Add width attribute
Image stretched/squashed Set both width and height Set only width OR height, not both

Key Takeaways

  • Use <img src="file.jpg" alt="description"> for images
  • ALT text is required for accessibility and SEO
  • Use <figure> and <figcaption> for captioned images
  • Always credit photos and check copyright
  • Embed videos from YouTube/Vimeo or use <video> tag
  • Use <audio> for podcasts and interviews
  • JPG for photos, PNG for graphics
  • Organize images in a separate "images" folder
