Create professional layouts for your journalism projects

Understanding Layout

Layout controls how elements are arranged on your page. Good layout makes content easy to read and navigate.

Display Property

The display property determines how an element behaves:

Value Behavior Common Elements
block Takes full width, starts new line h1, p, div
inline Only takes needed width, stays in line a, strong, em
inline-block Inline but accepts width/height Custom elements
none Hidden completely Hidden elements

Flexbox - Modern Layout Tool

Flexbox makes it easy to create rows and columns. Perfect for navigation menus, article grids, and more.

Basic Flexbox

Flexbox Container
.container {
    display: flex;
    gap: 20px;  /* Space between items */
}

.item {
    flex: 1;  /* Items grow equally */
}

Key Flexbox Properties

Flexbox Options
.flex-container {
    display: flex;

    /* Direction */
    flex-direction: row;  /* row, column */

    /* Horizontal alignment */
    justify-content: space-between;
    /* flex-start, center, space-around */

    /* Vertical alignment */
    align-items: center;
    /* flex-start, flex-end, stretch */

    /* Wrapping */
    flex-wrap: wrap;  /* nowrap, wrap */
}

TRY IT: Three-Column Layout

<style>
.three-columns {
    display: flex;
    gap: 20px;
}
.column {
    flex: 1;
    background: #f0f0f0;
    padding: 20px;
}
</style>

<div class="three-columns">

    <div class="column">Column 2</div>
    <div class="column">Column 3</div>
</div>

CSS Grid - Advanced Layouts

Grid creates complex two-dimensional layouts. Great for news homepages with multiple article cards.

Grid Layout
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;  /* 3 equal columns */
    gap: 20px;
}

/* Or with different sizes */
.grid-container {
    display: grid;
    grid-template-columns: 2fr 1fr;  /* 2/3 and 1/3 */
    gap: 20px;
}

Responsive Design Basics

Responsive design makes your site work on all screen sizes - phones, tablets, and desktops.

Media Queries

Mobile-First Approach
/* Mobile styles (default) */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet and up */
@media (min-width: 768px) {
    .container {
        width: 750px;
        padding: 20px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        width: 960px;
        padding: 40px;
    }
}
Mobile-First: Start with mobile styles, then add complexity for larger screens. Most news readers are on phones!

Key Takeaways

  • Use Flexbox for one-dimensional layouts (rows or columns)
  • Use Grid for two-dimensional layouts (rows AND columns)
  • Media queries make sites responsive
  • Start with mobile design, then scale up
  • Navigation menus use lists styled with Flexbox
  • Hover effects improve user experience
  • max-width and margin auto create centered containers
